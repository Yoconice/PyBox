# 文件时间修改工具

## 简介
这是一款跨平台的Python脚本工具，支持修改文件的**创建时间、修改时间、访问时间**（系统支持度有差异）。工具适配Windows、Linux、macOS系统，支持自定义时间格式，空参数时保留原始时间，满足文件时间属性的灵活管理需求。

## 核心功能
- ✅ 同时支持**创建/修改/访问**三个时间维度的修改
- ✅ 空参数自动保留对应时间的原始值
- ✅ 自定义时间格式（默认`YYYY-MM-DD HH:MM:SS`）
- ✅ 跨平台兼容（Windows/Linux/macOS）
- ✅ 完善的错误提示（文件不存在、权限不足等场景）

## 系统支持度说明
不同系统对文件时间的修改权限存在差异，核心支持情况：

| 系统       | 创建时间 | 修改时间 | 访问时间 | 备注                     |
|------------|----------|----------|----------|--------------------------|
| Windows    | ✅ 支持   | ✅ 支持   | ✅ 支持   | 需要依赖`pywin32`库      |
| Linux      | ❌ 不支持 | ✅ 支持   | ✅ 支持   | 内核限制，创建时间无法修改 |
| macOS      | ❌ 不支持 | ✅ 支持   | ✅ 支持   | 仅部分版本支持特殊修改   |

## 环境准备
### 1. Python版本
支持Python 3.6及以上（推荐3.8+，Python 3.12需搭配`pywin32>=306`）

### 2. 依赖安装（仅Windows）
Windows系统需安装`pywin32`库以支持时间修改：
```bash
pip install pywin32  # 自动安装兼容版本
# 若Python 3.12安装失败，指定版本：
pip install pywin32==306
```



```bash
# 示例配置（根据实际需求修改）
TARGET_FILE = r"E:\ProjectFiles\test.txt"  # 目标文件路径（用r避免反斜杠转义）
NEW_CREATE_TIME = "2025-01-01 09:00:00"    # 新创建时间（Windows有效）
NEW_MODIFY_TIME = "2025-01-01 10:00:00"    # 新修改时间
NEW_ACCESS_TIME = "2025-01-01 11:00:00"    # 新访问时间（可设为None表示不修改）
```
